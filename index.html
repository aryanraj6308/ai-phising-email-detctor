How It Works

A clean NLP pipeline from raw email text to a calibrated risk prediction.

01
Text Ingestion
Email text is received via paste or file upload. It's stripped, cleaned, and passed through a preprocessing pipeline.
02
TF-IDF Vectorization
The text is converted to numeric features using term frequency–inverse document frequency with 1-gram and 2-gram tokenization.
03
Logistic Regression
A balanced logistic regression classifier estimates the probability of phishing vs. safe for the given message vector.
04
Risk Output
Results are displayed with a probability score, risk tier, suspicious term list, and highlighted email preview.

Get Running in
Under 2 Minutes

Clone, install, train, run. No configuration files, no environment variables, no accounts needed.

1

Clone the Repository

git clone https://github.com/your-username/ai-phishing-email-detector.git
cd ai-phishing-email-detector
2

Create a Virtual Environment & Install Dependencies

python -m venv .venv
.venv\Scripts\activate          # Windows
# source .venv/bin/activate     # macOS/Linux
pip install -r requirements.txt
3

Train the Model

python train_model.py
4

Launch the Dashboard

streamlit run app.py
✦

Optional: Use a Kaggle Dataset for Better Accuracy

python train_model.py --data path/to/your_dataset.csv
